# Portfolio Assets

Place the following image files in the `assets/` folder:

- `assets/aws.png`         — AWS logo
- `assets/kubernetes.png`  — Kubernetes logo  
- `assets/terraform.png`   — Terraform logo
- `assets/docker.png`      — Docker logo
- `assets/jenkins.png`     — Jenkins logo
- `assets/github.png`      — GitHub logo

If images are missing, the skill cards will fallback to emoji icons automatically.

## Recommended image size: 80×80px, transparent PNG
## Source: Brand logos from official sites or https://devicon.dev
